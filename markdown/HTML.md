# HTML

## 语法

    **一些普通的行级元素可以直接嵌套入<em>markdown</em>中**  

    <em>怎么嵌套都行**加粗**</em>

    <p>块级元素需要前后加上空格，且内部无法使用**markdown**语法</p>

    <table>
      <tr>
        <td>表格</td>
        <td>表格</td>
        <td>表格</td>
      </tr>
    </table>


效果如下

>**一些普通的行级元素可以直接嵌套入<em>markdown</em>中**  
><em>怎么嵌套都行**加粗**</em>
><p>块级元素需要前后加上空格，且内部无法使用**markdown**语法</p>  
><table>
> <tr>
>   <td>表格</td>
>   <td>表格</td>
>   <td>表格</td>
> </tr>
></table>
